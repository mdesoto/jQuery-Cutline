Cutline is a jQuery plug-in forked from Captify originally written by Brian Reavis. It's only function is to display simple captions -- known as cutlines in the media world -- that transparently hover over the bottom portion of an image. Tested in Chrome 6.0.472, Internet Explorer 8, Firefox 3.6.10, Opera 9.64, and Safari 5.02. This plugin has been tested and confirmed working with earlier versions of the most popular browsers.
